## Speech-to-Text Transcription Tool 

This project is a simple Speech-to-Text application built using Python and Tkinter. It converts spoken audio file into text using the SpeechRecognition library and displays the output in a user-friendly graphical interface.

---

## Features
- Simple and clean GUI using Tkinter  
- Converts speech from WAV type audio files into text  
- Displays transcription within the application  
- Allows saving transcribed text as a `.txt` file  

---

## Technologies Used
- Python  
- Tkinter  
- SpeechRecognition  

---

## Audio Requirements 

For best results, the input audio file should meet the following conditions:

- Audio file format: WAV  
- Audio quality: Clear speech with minimal background noise  
- Language: English  
- Connection to internet is required.(it does not work offline)
